# Release

### :material-tag-outline: v0.0.2 - 26 May 2026
---

<br>

### :material-code-json: Schema Improvements (Machine Readable Results)
---

Improved API response consistency for AI agents, automation systems, and developer integrations.

- Standardized response structures across all endpoints
- Improved machine-readable JSON formatting
- Added explicit typing for key fields such as queries, metadata, and results
- Ensured predictable JSON keys for agents and integrations
- Added structured error objects with consistent error codes and messages

<br>

### :material-shield-lock-outline: API Usage Enforcement (Anti-Abuse Layer)
---

Strengthened API protection and abuse prevention systems.

- Added stricter request validation at the gateway layer
- Improved distributed rate-limit enforcement consistency
- Added early request rejection for invalid or abusive traffic
- Reduced unnecessary backend processing load from malformed requests

<br>

### :material-view-dashboard-outline: Dashboard Improvements (Minor Fixes)
---

Several dashboard stability and UI improvements were introduced.

- Fixed UI inconsistencies in dashboard data rendering
- Improved spacing and alignment in metric panels
- Corrected display issues affecting recent request logs
- Enhanced stability of the real-time activity feed under high request volume
